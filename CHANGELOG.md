## [0.1.0] - 2025-01-02

### Added
- Initial release
- 19 MCP tools for Willys grocery operations
- Stdio-based MCP server for Claude Code integration
- Next.js web interface for orders and offers
- SQLite-backed session management with 24h expiration
- Order caching system (80-90% API call reduction)
- Smart product matching with OpenAI embeddings + sqlite-vec
- Puppeteer-based authentication
- Full Swedish character support (ö, ä, å)
- CSRF token handling for secure API access
